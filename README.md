# 🏆 Placar Interativo

Projeto desenvolvido para auxiliar dinâmicas de competição em sala de aula, especialmente em oficinas com metodologias ativas.

## 📌 Sobre o Projeto

Este sistema funciona como um placar interativo com cronômetro para competições entre equipes.  
Foi desenvolvido com foco em atividades educacionais, proporcionando uma experiência visual dinâmica, intuitiva e envolvente.

## 🎯 Objetivo

Auxiliar na condução de atividades gamificadas durante a oficina:

**Matemática com Metodologias Ativas**  
Evento de **Acolhida dos Calouros 2026**

## ⚙️ Funcionalidades

- ⏱️ Cronômetro com cores dinâmicas:
  - Verde: acima de 50%
  - Amarelo: entre 50% e 25%
  - Vermelho: abaixo de 25%
  
- 🏆 Sistema de equipes:
  - Nome editável
  - Cor personalizada
  - Pontuação dinâmica

- 🎮 Interatividade:
  - Botões de +10 e -10 pontos
  - Destaque automático do líder
  - Sons ao pontuar e trocar liderança

- ⏸️ Controle de tempo:
  - Iniciar
  - Pausar / Retomar
  - Resetar

- 🧠 Interface visual:
  - Design moderno com animações
  - Feedback visual e sonoro

- 📊 Tela final:
  - "TIME IS OVER"
  - Pode ser ocultada/exibida com clique
